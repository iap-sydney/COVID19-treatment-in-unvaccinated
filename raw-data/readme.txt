The raw-data/ directory holds data that is read by scripts in processing/

Sub-directories can be created in raw-data/

raw-data/ should be ignored in .gitignore when data is sensitive and shouldn't be tracked in the git commit history

trt_data_doseagg.csv contains the data extracted from studies. Different doses of the same treatment have been aggregated.

bootstrap files contain bootstrapping results generated by the code so they don't need to be re-run every time.

If run_full_analysis = FALSE bootstrap data is loaded from raw-data directory

If run_full_analysis = TRUE bootstrapping code is run and new bootstrap data is saved in output directory and used from there.
